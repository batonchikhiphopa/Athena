# Athena

Editor-first daily reflection app with a hidden analytical engine.

## Project Layout

- `client/` - React/Vite frontend.
- `server/` - Express backend, SQLite access, extraction, analytics, insights.
- `shared/` - Reserved for contracts shared by frontend and backend.
- `migrations/` - SQLite migrations.
- `docs/` - Product and implementation notes.
- `data/` - Local runtime data.
